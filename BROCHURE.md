# sim-runtime

In one line: the working machinery of a running SIM program -- names, dispatch, control, and a dozen languages to write it in.

## What it gives you

The shared plumbing every library installs through, name binding and namespaces, kind-based dispatch, pattern matching that warns on a missed case, tracked in-place mutation, logic queries, and language surfaces from Common Lisp and Scheme to Lua, Ruby, Julia, Clojure, and Prolog -- plus the batteries-included bundle that makes SIM useful the moment it starts.
